Saldo Bip!

Ingresa el número de tu tarjeta Bip! para obtener su saldo.

Qué

Este es un demo de una aplicación web que combina un end-point para consultar el saldo de tarjetas Bip! y una simple interfaz que consume dicho end-point.

Código fuente

Cómo

Tecnologías utilizadas:

  • Node.JS
  • Express
  • Request
  • Cheerio

End-point

Para consumir el end-point solo basta enviar un request GET que incluya el número de la tarjeta Bip!, como se indica a continuación:

curl http://localhost:8080/api/balance/1234
